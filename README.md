# 🍱 MatPreppern

MatPreppern is a Progressive Web App (PWA) built to make meal prepping easier. The goal is to help users discover recipes, plan meals for the next 1-14 days, adjust portions automatically, and generate a shopping list based on their meal plan.

The project is built with vanilla JavaScript and focuses on creating a fast, responsive, and installable web application without using frameworks.

## Features

- 🍗 Browse meal prep recipes
- 🔍 Search and filter recipes
- 📊 View calories and macronutrients
- 🍽️ Automatic portion scaling
- ➕ Add your own recipes
- 📅 Weekly meal planner
- 🛒 Automatically generated shopping list
- 📱 Installable as a Progressive Web App (PWA)
- 💾 Offline support through Service Workers

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage
- Service Workers
- Web App Manifest

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/matpreppern.git
```

Open the project with Live Server or any local web server.
